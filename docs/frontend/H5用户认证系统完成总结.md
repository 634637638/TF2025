## 🎯 核心功能

### 用户注册
- 姓名、手机号、密码注册
- 手机号格式验证（1[3-9]xxxxxxxxx）
- 密码长度验证（至少6位）
- 自动标记为"H5用户"
- 注册成功自动登录

### 用户登录
- 手机号+密码登录
- 密码错误提示
- 未设置密码提示（针对老客户）
- 生成30天有效token

### 我的中心
- **未登录状态**: 显示登录/注册按钮
- **已登录状态**:
  - 显示用户信息（姓名、手机号、头像）
  - 我的订单入口（跳转到订单查询）
  - 购买记录弹窗（显示历史销售记录）
  - 登出功能

### 数据关联
- H5_orders表: 通过 customer_phone 关联
- sales表: 通过 customer_id 关联
- 支持查看用户的所有购买历史

## 🔒 安全特性

1. **密码加密**: bcrypt加密，salt rounds = 10
2. **Token管理**: UUID token，30天有效期
3. **请求认证**: Bearer Token在Authorization header中传递
4. **输入验证**: 前后端双重验证
5. **SQL注入防护**: 使用参数化查询

## 🚀 使用流程

### 新用户
1. 访问 `/m/my` 页面
2. 点击"注册账号"
3. 填写姓名、手机号、密码
4. 注册成功后自动登录

### 老用户
1. 访问 `/m/my` 页面
2. 点击"立即登录"
3. 输入手机号和密码
4. 登录成功后查看个人信息和订单
