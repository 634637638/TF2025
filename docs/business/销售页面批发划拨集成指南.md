# 销售页面批发/划拨按钮集成指南

## 目标

在销售页面的按钮区域添加"批发给同行"和"代供应商划拨"按钮，位置在对库按钮前面。

## 按钮布局设计

```
当前布局：
[对库] [图文] [表格]

新布局：
[批发给同行] [代划拨] | [对库] [图文] [表格]
```

## 实施步骤

### 步骤1：在 SalesView.vue 中添加组件导入

在 `<script setup>` 部分添加：

```typescript
import WholesaleModal from '@/components/WholesaleModal.vue'
```

### 步骤2：添加状态变量

```typescript
// 批发/划拨对话框
const showWholesaleModal = ref(false)
const wholesaleMode = ref<'wholesale' | 'proxy'>('wholesale')
```

### 步骤3：修改按钮区域

将现有的按钮区域从：

```vue
<div class="view-controls">
  <div class="view-toggle">
    <el-button @click="viewMode = 'summary'" ...>对库</el-button>
    <el-button @click="viewMode = 'grid'" ...>图文</el-button>
    <el-button @click="viewMode = 'table'" ...>表格</el-button>
  </div>
</div>
```

改为：

```vue
<div class="view-controls">
  <!-- 批发/划拨按钮组 -->
  <div class="wholesale-actions" v-if="selectedPhones.length > 0">
    <el-button
      type="success"
      size="small"
      @click="handleWholesale"
      :disabled="!canWholesale"
    >
      <i class="fas fa-boxes"></i>
      <span>批发给同行</span>
    </el-button>
    <el-button
      type="warning"
      size="small"
      @click="handleProxyTransfer"
      :disabled="!canProxy"
    >
      <i class="fas fa-exchange-alt"></i>
      <span>代划拨</span>
    </el-button>
    <el-divider direction="vertical" />
  </div>

  <!-- 视图切换按钮组 -->
  <div class="view-toggle">
    <el-button @click="viewMode = 'summary'" ...>对库</el-button>
    <el-button @click="viewMode = 'grid'" ...>图文</el-button>
    <el-button @click="viewMode = 'table'" ...>表格</el-button>
  </div>
</div>
```

### 步骤4：添加方法

```typescript
// 批发给同行
const handleWholesale = () => {
  wholesaleMode.value = 'wholesale'
  showWholesaleModal.value = true
}

// 代供应商划拨
const handleProxyTransfer = () => {
  wholesaleMode.value = 'proxy'
  showWholesaleModal.value = true
}

// 检查是否可以批发
const canWholesale = computed(() => {
  return selectedPhones.value.length > 0 &&
    selectedPhones.value.every(p => p.status === 'in_stock')
})

// 检查是否可以代划拨
const canProxy = computed(() => {
  return selectedPhones.value.length > 0 &&
    selectedPhones.value.every(p => p.status === 'in_stock')
})

// 批发/划拨成功回调
const handleTransferSuccess = () => {
  clearBatchSelection()
  loadAvailablePhones()
  ElMessage.success('操作成功')
}
```

### 步骤5：添加对话框组件

在模板最后添加：

```vue
<!-- 批发/划拨对话框 -->
<WholesaleModal
  v-model:visible="showWholesaleModal"
  :mode="wholesaleMode"
  :phone-ids="selectedPhones.map(p => p.id)"
  :phones="selectedPhones"
  @success="handleTransferSuccess"
/>
```
