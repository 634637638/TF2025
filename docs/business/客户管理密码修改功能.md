# 客户管理密码修改功能

## 功能概述
在客户管理页面的编辑模态框中添加了密码修改功能，允许管理员为H5商城用户设置或修改登录密码。

## 前端修改

### 文件：`/frontend/src/views/customers/CustomersView.vue`

#### 1. 表单数据结构更新
在 `customerForm` 中添加了密码相关字段：
```typescript
const customerForm = ref({
  // ... 其他字段
  changePassword: false,    // 是否启用密码修改
  password: '',              // 新密码
  confirmPassword: ''        // 确认密码
})
```

#### 2. UI组件添加
在编辑模态框中添加了密码设置区域：
- 仅在编辑模式（`modalMode === 'edit'`）时显示
- 包含启用密码修改的复选框
- 新密码输入框（带显示/隐藏功能）
- 确认密码输入框
- 友好的提示文字

#### 3. 表单验证更新
在 `saveCustomer` 函数中添加了密码验证：
- 密码长度验证（至少6位）
- 确认密码一致性验证
- 只有勾选"启用密码修改"时才进行密码验证

#### 4. 数据提交逻辑
- 编辑模式且勾选修改密码时：包含密码字段
- 编辑模式但未勾选修改密码时：移除所有密码相关字段
- 新增模式：不包含密码字段（新增用户不需要设置密码）

## 后端修改

### 文件：`/backend/src/routes/customers.js`

#### 1. 添加bcrypt导入
```javascript
const bcrypt = require('bcrypt');
```

#### 2. 更新请求参数解构
在更新客户API中添加了 `password` 字段：
```javascript
const {
  // ... 其他字段
  password
} = req.body;
```

#### 3. 添加密码处理逻辑
```javascript
// 处理密码更新（如果提供）
if (password !== undefined && password !== null && password !== '') {
  // 验证密码长度
  if (password.length < 6) {
    return ApiResponse.badRequest(res, '密码至少需要6位');
  }
  // 加密密码
  const hashedPassword = await bcrypt.hash(password, 10);
  updateData.password = hashedPassword;
}
```

## 使用说明

### 管理员操作流程
1. 访问客户管理页面 (`/customers`)
2. 点击要编辑的客户行的"编辑"按钮
3. 在编辑模态框中找到"密码设置"部分
4. 勾选"启用密码修改"复选框
5. 输入新密码（至少6位）
6. 再次输入密码确认
7. 点击"保存"按钮

### 用户使用流程
1. 用户访问H5商城 (`/m/my`)
2. 点击"立即登录"
3. 使用管理员设置的密码登录
4. 登录成功后可查看订单和购买记录

## 安全特性

1. **密码加密**：使用bcrypt加密，salt rounds = 10
2. **密码验证**：
   - 前端：至少6位，两次输入必须一致
   - 后端：至少6位验证
3. **条件更新**：只有明确勾选"启用密码修改"时才更新密码
4. **权限控制**：需要客户管理编辑权限才能修改密码

## 测试场景

### 测试用例1：正常修改密码
1. 选择一个客户，点击编辑
2. 勾选"启用密码修改"
3. 输入密码：123456
4. 确认密码：123456
5. 点击保存
**预期结果**：密码更新成功，客户可以使用新密码登录

### 测试用例2：密码不一致
1. 选择一个客户，点击编辑
2. 勾选"启用密码修改"
3. 输入密码：123456
4. 确认密码：1234567
5. 点击保存
**预期结果**：提示"两次输入的密码不一致"

### 测试用例3：密码过短
1. 选择一个客户，点击编辑
2. 勾选"启用密码修改"
3. 输入密码：123
4. 确认密码：123
5. 点击保存
**预期结果**：提示"新密码至少需要6位"

### 测试用例4：不修改密码
1. 选择一个客户，点击编辑
2. 不勾选"启用密码修改"
3. 修改其他字段（如姓名）
4. 点击保存
**预期结果**：其他字段更新成功，密码保持不变

## 注意事项

1. **新增客户**：新增客户时不会显示密码设置区域
2. **已有密码**：如果客户已有密码，修改操作会替换旧密码
3. **H5登录**：只有设置了密码的客户才能在H5商城登录
4. **老客户**：对于在sales表中存在但未设置密码的客户，登录时会提示联系管理员
