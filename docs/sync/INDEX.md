## 使用场景

1. **本地到云端**：将本地数据同步到云端数据库
2. **云端到本地**：从云端数据库获取数据到本地
3. **智能合并**：自动识别并合并重复数据
4. **批量导入**：从 Excel/CSV 批量导入业务数据

## 注意事项

- 同步前请**备份重要数据**

- 检查数据库权限配置
