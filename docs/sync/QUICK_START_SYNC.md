# 远程数据同步 - 快速开始

## 🚀 5分钟快速上手

### 1. 访问功能
```
打开浏览器访问: http://localhost:5176/data-optimization
点击"远程数据同步"标签
```

### 2. 连接数据库
```
主机地址: 例如 192.168.1.100
端口: 3306（默认）
用户名: 数据库用户名
密码: 数据库密码
数据库名: 源数据库名
点击"连接数据库"
```

### 3. 选择表
```
左侧: 选择外部数据库的表（例如：old_phones）
右侧: 选择本地数据库的表（例如：phones）
```

### 4. 配置映射
```
1. 点击"智能匹配"按钮
2. 系统自动配置字段映射
3. 检查并调整映射关系
4. 选择同步模式（推荐：upsert）
5. 选择匹配字段（例如：imei）
```

### 5. 预检查
```
点击"开始检查"
查看统计信息：
- 总数据量
- 新数据数量
- 需要更新的数据数量
```

### 6. 执行同步
```
点击"开始同步"
实时查看进度：
- ✅ 自动创建品牌、型号、颜色等
- ✅ 插入新数据
- ✅ 更新已存在数据
- ✅ 显示详细统计
```

## 📋 常见使用场景

### 场景1：从老系统迁移到新系统
```
源表: old_phones
目标表: phones
匹配字段: imei
同步模式: upsert
```
✅ 结果：自动创建所有品牌、型号等基础数据，同步手机记录

### 场景2：定期同步客户信息
```
源表: remote_customers
目标表: customers
匹配字段: phone
同步模式: upsert
```
✅ 结果：自动创建新客户并生成会员号，更新已有客户信息

### 场景3：多店数据汇总
```
源表: branch1_phones
目标表: phones
匹配字段: imei
同步模式: upsert
```
✅ 结果：自动创建分店信息，汇总所有手机数据
