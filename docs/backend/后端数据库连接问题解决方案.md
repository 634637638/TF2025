# 后端数据库连接问题解决方案

## 问题诊断

### 当前状态
后端服务器无法启动，原因是云端数据库连接超时：
```
数据库连接池创建失败: ETIMEDOUT connect ETIMEDOUT 59.63.121.218:3306
```

### 数据库配置
- 主机: v4.cn9527.cn
- 端口: 3306
- 数据库: TF2025
- 用户: TF2025

## 可能的原因

1. **网络问题**：本地到云端数据库服务器网络不通
2. **防火墙/安全组**：云端数据库安全组未允许本地IP访问
3. **数据库服务状态**：云端数据库服务未启动或异常
4. **DNS解析**：域名解析问题
5. **连接数限制**：数据库连接数达到上限

## 解决方案

### 方案1：检查网络连接 ✅
```bash
# 测试数据库端口是否可达
nc -zv v4.cn9527.cn 3306

# 或使用 telnet
telnet v4.cn9527.cn 3306

# 检查DNS解析
nslookup v4.cn9527.cn

# 检查路由追踪
traceroute v4.cn9527.cn
```

### 方案2：检查云端数据库配置 🔧
需要登录云端数据库管理控制台检查：

**安全组设置**：
- 确保允许本地IP访问3306端口
- 如果不确定，可以临时添加 `0.0.0.0/0` 允许所有IP访问

**数据库状态**：
- 确认数据库服务正在运行
- 检查连接数限制
- 查看数据库日志

### 方案3：使用本地数据库 🏠（推荐临时方案）
修改 `/backend/.env` 文件，使用本地MySQL数据库：

```env
# 本地数据库配置
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=TF2025
DB_CONNECTION_LIMIT=20
```

### 方案4：增加连接超时时间 ⏱️
修改 `/backend/src/config/database.js`，增加超时时间：

```javascript
const poolConfig = {
  ...config.db,
  connectTimeout: 300000,  // 增加到300秒 (5分钟)
  acquireTimeout: 300000,  // 获取连接超时
  // ... 其他配置
};
```

### 方案5：跳过CSRF清理任务 🚀
修改 `/backend/src/routes/csrf.js`，禁用自动清理任务：

```javascript
// 注释掉或删除自动清理任务
// setTimeout(cleanupAllExpiredTokens, 1800000); // 30分钟
```

## 验证数据库连接

### 测试脚本
创建 `/backend/test-db.js`：
```javascript
const mysql = require('mysql2/promise');
require('dotenv').config();

async function testConnection() {
  try {
    console.log('测试数据库连接...');
    console.log(`主机: ${process.env.DB_HOST}`);
    console.log(`端口: ${process.env.DB_PORT}`);
    console.log(`数据库: ${process.env.DB_NAME}`);

    const connection = await mysql.createConnection({
      host: process.env.DB_HOST,
      port: process.env.DB_PORT,
      user: process.env.DB_USER,
      password: process.env.DB_PASSWORD,
      database: process.env.DB_NAME,
      connectTimeout: 10000
    });

    const [rows] = await connection.execute('SELECT 1');
    console.log('✅ 数据库连接成功！');

    await connection.end();
    process.exit(0);
  } catch (error) {
    console.error('❌ 数据库连接失败:', error.message);
    process.exit(1);
  }
}

testConnection();
```

运行测试：
```bash
node test-db.js
```
