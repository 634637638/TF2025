# 📖 书签工具使用详细教程

## 什么是书签工具？

书签工具是一个特殊的浏览器书签，点击后会自动运行一段 JavaScript 代码，帮你从天翼云盘页面提取登录 Cookie。

## 🎯 完整使用流程

### 步骤 1：打开工具页面

在浏览器地址栏输入：
```
http://localhost:5176/get-ctyun-cookie.html
```

或者在云备份配置页面点击"点击打开 Cookie 获取工具"链接。

---

### 步骤 2：添加书签到书签栏

#### 2.1 确保书签栏可见

**如果看不到书签栏**，按快捷键显示：
- **Chrome/Edge**: `Ctrl + Shift + B`（Mac: `Cmd + Shift + B`）
- **Firefox**: `Ctrl + Shift + B`
- **Safari**: `Cmd + Shift + B`

书签栏通常在地址栏下方，显示你保存的网站快捷方式。

#### 2.2 拖拽书签

在工具页面找到蓝色的链接按钮：**🔖 拖我到书签栏**

**操作方法**：
1. 鼠标移到蓝色链接上
2. **按住鼠标左键不放**
3. **拖动**鼠标到浏览器顶部的书签栏
4. 看到书签栏出现一个占位符时，**松开鼠标**
5. 书签添加成功！

**成功标志**：
- 书签栏会出现一个名为"🔖 拖我到书签栏"的书签
- 你可以右键点击书签，选择"编辑"来修改名称（比如改成"获取天翼云Cookie"）

---

### 步骤 3：登录天翼云盘

1. 打开新标签页
2. 访问：`https://cloud.189.cn`
3. 使用你的手机号和密码**正常登录**
4. 确保能看到云盘主页（显示你的文件列表）

**重要**：必须先登录成功，书签才能提取到 Cookie！

---

### 步骤 4：使用书签提取 Cookie

在天翼云盘页面（确保 URL 是 cloud.189.cn）：

1. **点击**书签栏中刚才添加的书签
2. 页面会弹出一个白色窗口
3. 窗口中显示提取到的 Cookie（一长串文本）
4. 点击窗口中的"📋 复制"按钮
5. 看到"✅ 已复制到剪贴板！"提示

**如果没有弹出窗口**：
- 检查是否在 cloud.189.cn 页面
- 检查是否已经登录
- 尝试刷新页面后再点击书签

---

### 步骤 5：配置到系统

1. 回到你的云备份配置页面
2. 找到"Cookie"输入框（在密码输入框下方）
3. 在输入框中**右键 → 粘贴**（或按 `Ctrl + V`）
4. 确认粘贴的内容类似：
   ```
   COOKIE_LOGIN_USER=xxx; SESSION_KEY=xxx; UID=xxx
   ```
5. 点击"保存并连接"按钮
6. 等待几秒，看到"✅ 连接成功！"提示

## ❓ 常见问题

### Q1: 拖拽书签时没有反应？

**解决方法**：
- 确保书签栏已显示（按 `Ctrl + Shift + B`）
- 尝试用鼠标**右键点击**蓝色链接 → 选择"添加到书签"或"收藏此链接"
- 有些浏览器需要拖到书签栏的**空白区域**

### Q2: 点击书签后没有弹窗？

**可能原因**：
1. **不在天翼云盘页面**：必须在 cloud.189.cn 页面使用
2. **未登录**：需要先登录天翼云盘
3. **浏览器阻止了弹窗**：检查地址栏右侧是否有弹窗拦截图标

**解决方法**：
- 确保在 cloud.189.cn 页面
- 刷新页面后重新登录
- 允许该网站的弹窗

### Q3: 提示"未找到登录 Cookie"？

**原因**：
- 还没有登录天翼云盘
- 登录已过期

**解决方法**：
1. 重新登录天翼云盘
2. 确保能看到云盘主页
3. 再次点击书签

### Q4: 复制的 Cookie 粘贴后无法连接？

**检查项**：
1. Cookie 是否完整（应该包含 COOKIE_LOGIN_USER、SESSION_KEY 等）
2. 是否有多余的空格或换行
3. Cookie 是否已过期（重新获取）

### Q5: 书签可以重复使用吗？

**答案**：可以！
- 书签只需添加一次
- 每次天翼云盘登录后都可以用书签获取新 Cookie
- Cookie 过期后，重新登录天翼云盘，再用书签获取即可

## 💡 高级技巧

### 技巧 1：修改书签名称

右键点击书签 → 编辑 → 修改名称为"获取天翼云Cookie"，更容易识别。

### 技巧 2：快捷键访问

某些浏览器支持给书签设置快捷键，可以更快速地调用。

### 技巧 3：多账号管理

如果有多个天翼云账号：
1. 登录账号 A，用书签获取 Cookie A
2. 退出登录，登录账号 B，获取 Cookie B
3. 在系统中切换使用不同的 Cookie

---

## 🔒 安全说明

- Cookie 包含登录凭证，请勿分享给他人
- 书签代码是开源的，可以查看源码确认安全性
- Cookie 仅存储在你的本地服务器
- 建议定期更换 Cookie（重新登录获取）
